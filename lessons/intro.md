# Intro

Welcome to the ABC classroom repo. This repo is the learning and examples space
for Random Knights contributors. It is intentionally separate from the
production app so students can inspect scripts, practice documentation, and
build small local exercises without changing runtime systems.

## How To Use This Repo

1. Read a lesson in `lessons/`.
2. Complete the matching lab in `labs/`.
3. Copy any script you want to experiment with into a local `workspace/`
   folder.
4. Keep generated files out of git.
5. Ask for review before running scripts that touch the file system, network,
   credentials, or system settings.

## Safety Rules

- Do not commit secrets, API keys, credentials, or private URLs.
- Do not commit `ffmpeg.exe`, `ffprobe.exe`, generated videos, or local tool
  caches.
- Treat cleanup/system scripts as static review material unless a mentor
  explicitly approves execution.
- Keep production Rand0m app work out of this classroom repo.

## Suggested Path

Start with `01-classroom-orientation`, then move through the numbered lessons
and labs in order.
