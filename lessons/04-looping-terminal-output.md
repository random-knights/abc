# Lesson 04: Looping Terminal Output

## Goal

Learn how loops and display effects can make a terminal feel interactive while
still remaining easy to inspect.

## Concepts

- loops
- delays
- color commands
- screen clearing
- exit paths

## Setup

Use this asset:

- `assets/batch/RAINBOW MATRIX.bat`

Read the script before running it.

## Steps

1. Locate the main loop.
2. Identify commands that change the terminal display.
3. Find how the script exits, if it exits.
4. Write down the difference between visual output and data output.
5. Explain why infinite loops need a clear stop instruction.

## Challenge

Design a calmer version of the same idea: fewer colors, shorter output, and a
clear quit instruction.

## Expected Outcome

You can recognize terminal animation patterns and describe how to keep them
safe for beginners.

## Cleanup / Reset

Close the terminal window if a copied script keeps running. Do not commit edited
copies.
