# Lab 05: Turn-Based Game State

## Goal

Convert game script behavior into testable rules.

## Concepts

- state
- valid move
- invalid move
- win condition
- reset

## Setup

Copy these into `workspace/game-state/`:

- `assets/batch/TIC-TAC-TOE.bat`
- `assets/batch/POKEMON.bat`

## Steps

1. Pick one script.
2. Identify variables that represent current state.
3. Trace one valid path.
4. Trace one invalid path.
5. Write three test cases: happy path, invalid input, reset/end.

## Challenge

Write one Cucumber-style scenario for a win or completion state.

## Expected Outcome

You can transform script behavior into structured test cases.

## Cleanup / Reset

Delete `workspace/game-state/`.
