# Lab 01: Classroom Orientation

## Goal

Build a local map of the ABC classroom repo and practice safe workspace setup.

## Concepts

- folder inventory
- ignored workspace folders
- relative file paths
- classroom source control hygiene

## Setup

Create a local ignored folder:

```text
workspace/orientation/
```

## Steps

1. List the repo folders.
2. Create a text file in `workspace/orientation/notes.txt`.
3. Write one line for `lessons`, `labs`, and `assets`.
4. Add one note explaining why `workspace/` is ignored.
5. Check `git status` and confirm your notes are not staged.

## Challenge

Add a fourth line describing what should never be committed from a lab.

## Expected Outcome

You have a local note that explains the repo structure, and git does not track
your workspace output.

## Cleanup / Reset

Delete `workspace/orientation/` when finished.
