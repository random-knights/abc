# Labs

Experimental AI tools and prototypes by Random Knights.

## Tools

### job-scorer/
AI-powered job description scorer. Paste a job description plus your resume
and cover letter for an instant personalized match analysis.
See job-scorer/README.md for setup.

### nonprofit-mailer/
AI-powered email campaign generator for mission-driven nonprofits.
Full 5-step pipeline from subject lines to eval-gated output.
See nonprofit-mailer/README.md for setup.

Both tools require an Anthropic API key from console.anthropic.com.
