# Lesson 01: Classroom Orientation

## Goal

Understand how the ABC classroom repo is organized and how to work safely with
lessons, labs, and assets.

## Concepts

- Repo structure
- Relative paths
- Source assets versus generated outputs
- Safe local workspace habits
- Documentation-first learning

## Setup

Open the repo root:

```text
C:\Projects\qa-kitt\abc
```

Review these folders:

- `lessons/`
- `labs/`
- `assets/batch/`
- `assets/mp3/`
- `assets/python/`

## Steps

1. Read `README.md`.
2. Open `lessons/intro.md` and `labs/sandbox.md`.
3. List the files under `assets/`.
4. Pick one batch, one Python, and one MP3 asset.
5. Write down what each asset appears to be for.

## Challenge

Create a small map of the repo:

```text
abc
  lessons -> explanation
  labs -> hands-on exercises
  assets -> reusable classroom samples
```

Add one sentence explaining why generated files should stay out of git.

## Expected Outcome

You can explain where learning content lives, where sample assets live, and why
local outputs should stay in ignored workspaces.

## Cleanup / Reset

No cleanup is required. If you created notes, keep them outside the repo or in a
local `workspace/` folder.
