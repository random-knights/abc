# Lab 03: Randomness And Input Validation

## Goal

Review random-value scripts and design better validation copy.

## Concepts

- random output
- input validation
- practice-only generated values
- user messaging

## Setup

Copy these into `workspace/randomness/`:

- `assets/batch/GUESSING GAME.bat`
- `assets/batch/PASSWORD GENERATOR.bat`

## Steps

1. Find every `%random%` use.
2. Write down the generated value shape.
3. Enter three invalid options on paper and predict script behavior.
4. Draft a safer error message.
5. Draft practice-only helper text for the password sample.

## Challenge

Write one Given/When/Then scenario for invalid input.

## Expected Outcome

You can turn a simple script behavior into validation requirements and a test
idea.

## Cleanup / Reset

Delete `workspace/randomness/`. Do not save generated passwords.
