# Lab 04: Looping Terminal Output

## Goal

Inspect terminal animation behavior and design a safer exit path.

## Concepts

- loops
- terminal effects
- exit instructions
- user control

## Setup

Copy this into `workspace/terminal-loop/`:

- `assets/batch/RAINBOW MATRIX.bat`

## Steps

1. Find the loop label.
2. Find commands that change terminal appearance.
3. Write the keyboard command you would use to stop a running script.
4. Design a menu option called `Exit`.
5. Add the design as comments in the copied file.

## Challenge

Create a plain-English acceptance criterion for "the user can stop the loop".

## Expected Outcome

You can review visual terminal scripts without losing sight of user control and
safe stopping behavior.

## Cleanup / Reset

Delete `workspace/terminal-loop/`.
