# Lab 02: Batch Command Basics

## Goal

Trace a small batch script and rewrite one prompt in a copied file.

## Concepts

- copying assets safely
- labels
- prompts
- branches

## Setup

Copy these into `workspace/batch-basics/`:

- `assets/batch/CALCULATOR.bat`
- `assets/batch/GUESSING GAME.bat`

## Steps

1. Open the copied `CALCULATOR.bat`.
2. Mark each prompt and each branch.
3. Change one prompt to be clearer.
4. Open the copied `GUESSING GAME.bat`.
5. Write a simple flow: start, guess, compare, result.

## Challenge

Add a comment above one confusing line in the copied file. Do not edit the
source asset.

## Expected Outcome

You can explain how beginner batch scripts receive input and choose the next
step.

## Cleanup / Reset

Delete `workspace/batch-basics/`.
