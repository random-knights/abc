# Lab 07: Ambient Asset Inventory

## Goal

Create a lightweight manifest for the classroom MP3 assets.

## Concepts

- asset cataloging
- relative paths
- media readiness
- non-duplication

## Setup

Use `assets/mp3/`.

Create:

```text
workspace/ambient-assets/manifest.md
```

## Steps

1. List all MP3 files.
2. Add a short theme for each file.
3. Pick three files for a focus preset.
4. Add one limitation note.
5. Confirm you did not copy or rename the MP3 files.

## Challenge

Add a column called `best classroom use` and fill it for every asset.

## Expected Outcome

You have a local asset manifest that references files by path instead of
duplicating media.

## Cleanup / Reset

Delete `workspace/ambient-assets/`.
